mStats

Instruction for installation

To install old versions directly from the webpage, run the following command by replacing #.#.# with version of your desire. 

install.packages("https://myominnoo.github.io/pages/r/mStatsLite_#.#.#.tar.gz", repos = NULL, type = "source")

For example: 

install.packages("https://myominnoo.github.io/pages/r/mStats_3.4.0.tar.gz", repos = NULL, type = "source")

Another way is to download .tar.gz file and install from RStudio.
