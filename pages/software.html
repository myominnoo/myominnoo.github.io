mStats: merry Statistics - An R Package by Myo

mStats is a package I write to help me in my daily tasks of R data management and analysis.
Here is how to install the package.


Installation

mStats package is hosted on Github. To install it, you can do it from R using install_github().


First Step: mStats package depends on ggplot2 package as I like its graph features. So install "ggplot2" package. 
install.packages("ggplot2")
Second Step: install "devtools" package. 
install.packages("devtools")
Third Step: load the package. 
library(devtools)
Finally, run the command. 
install_github("myominnoo/mStats")
